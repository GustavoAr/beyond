What is BEYONDJS ?

BeyondJS is a framework for building web application that can be easily packaged and deployed on web and mobile. To achieve this goal, BeyondJS solves:

  • module bundler, package your modules.
  • load on demand of html, css and javascript.
  • realtime ready.
  • single page.
  • offline ready.
  • multilanguage.
  • mustache ready for browser-side html templating.

BeyondJS uses the latest in HTML5 technology, allows the creation of web applications that can run offline, allowing synchronize resumed once the connection. Create once, distribute anywhere, is one of the goals of BeyondJS, help you create applications that can easily be ported to different mobile operating systems, desktop and still function as a web application in the browser. It allows you to download resources on demand, which means faster browsing and by using features like HTML5 application cache makes more fluid navigation recurring users of the web application. Allows use of third-party libraries, where all the logic depends on these and you just have to worry about the implementation and customization within your application. For this use of cross domain technologies and websockets.

In order to help you to organize your resources and javascript code, BeyondJS offer a modules development approach, that implies that you will handle your code in a similar fashion as you develop desktop applications.

BeyondJS integrates Socket.IO, {{ Mustache.js }} templating, RequireJS for load on demand of resources and of course, jQuery.

HTML5 Technology

The latest in HTML5 technology to drive the web applications to a whole new level of feature to improve the user experience and take it to a "like native" sense.

ES6 Ready

Load on demand of assets, multiple sockets connections, offline ready and multi-languages are some of the great features of JavaScript and its libraries behind BeyondJS.

Module Bundler

BeyondJS take modules, libraries and static files that are been used in the development and apply a process of concatenate, minify and packaging to make it ready to distribution.
